🤟 PSL Sign Language Translator

A real-time Pakistan Sign Language (PSL) Translator Web App using Machine Learning and MediaPipe. It detects hand gestures via webcam and converts them into text.

🚀 Features
🎥 Real-time gesture detection
✋ MediaPipe hand tracking
🤖 ML-based prediction
🔤 Gesture to text output
📊 Custom .npy dataset
🧠 Tech Stack
Python
MediaPipe
TensorFlow / Keras
OpenCV
NumPy
⚙️ Installation
git clone https://github.com/your-username/psl-translator.git
cd psl-translator
pip install -r requirements.txt
python app.py
